# aoc-2023

Advent of Code 2023

## Getting started
To install dependencies:

```bash
bun install
```

To run:

```bash
bun run day<number>/index.ts
```

To create files for a new day:
```bash
bun run newday
```

This project was created using `bun init` in bun v1.0.15. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
